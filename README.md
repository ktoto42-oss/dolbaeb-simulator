# Dolbaeb simulator

---

## Геймплейный концепт

Вы управляете персонажем, чья жизнь зациклена в бесконечном, мрачном марафоне зависимости. У вас есть три постоянно изменяющиеся шкалы состояния:
1. **High (Кайф)** - падает каждую секунду. Если упадет слишком низко - начнутся «отходы». Если поднимется выше 70% - вас захлестнет паранойя.
2. **Paranoia (Паранойя)** - растет, когда вы слишком сильно «вштырены» или когда кайф падает до критического минимума. При паранойе >70% мир вокруг начинает тревожно багроветь.
3. **Energy (Энергия)** - тратится на любые активности. Если энергии меньше 20%, персонаж физически замедляется и начинает еле волочить ноги.

### Игровой цикл:
* **Дома (Квартира):** Маленькое, безопасное (пока что) пространство. Здесь можно заняться «важными делами» - например, рассматривать **Ковер** (клавиша `E`). Это тратит энергию, но помогает временно сбить шкалу паранойи.
* **На улице (Город):** Огромная локация (переход по клавише `Space`). Здесь вам нужно рыскать по кустам в поисках заветного **клада**. Каждый найденный клад мгновенно восстанавливает Кайф и Энергию на 100%, перезапуская этот безумный цикл.

## 🛠️ Архитектура и технологии

Проект написан с нуля, что делает код максимально прямолинейным, понятным для новичков и чистым выражением классического игрового цикла (`Game Loop`).

* **Язык:** Rust 🦀
* **Графический движок:** [Macroquad](https://macroquad.rs/).
* **Управление:** * `W`, `A`, `S`, `D` - Перемещение
  * `Space` - Выйти на улицу / Вернуться домой (Смена локаций с динамическим клэмпингом границ)
  * `E` - Взаимодействовать с ковром (только дома, при Кайфе > 30%)
